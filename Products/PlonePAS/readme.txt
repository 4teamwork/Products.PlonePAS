PlonePAS 

  an integration of PluggableAuthService (PAS) into Plone. 

 Status

  Seriously Bleeding Edge Development

 Installation (FOR DEVELOPERS ONLY)

  add a plone instance without a user folder and use the quickinstaller
  to install this product into the portal.

 Migration

  Some attempts have been made to try and offer a seamless upgrade
  for sites using GRUF, via wrapping GRUF as a PAS plugin.
